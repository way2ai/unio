# Unio Documentation

These documents describe the Unio system after the from-zero refactor.

Recommended reading order:

- [Get Started](get-started.md): build, run, use the hybrid CLI, tools,
  approvals, skills, resume, trace, and tests.
- [Architecture](architecture.md): runtime layers, main execution flow, crate
  boundaries, and current architecture contracts.
- [Hybrid File References](hybrid-file-references.md): interactive `@path`
  suggestions and workspace indexing.
- [Hybrid Slash Commands](hybrid-slash-commands.md): interactive `/` command
  suggestions and completion behavior.
- [Hybrid Input Editing](hybrid-input-editing.md): cursor movement, delete
  shortcuts, and multiline prompt behavior.
- [Target Architecture](target-architecture.md): long-term agent, model, tools,
  security, storage, observability, and UX goals.
- [Download](download.md): how to obtain and build Unio.
- [Site Operations](site-operations.md): local preview, CI, and GitHub Pages.
- [Decisions](decisions/0001-start-from-zero.md): architecture decision records.

Keep docs and implementation synchronized. Update `PLAN.md`, `README.md`,
`CHANGE.md`, and the relevant `docs/` file when behavior changes.
