# 混合斜杠命令

混合 CLI 将 `/` 作为交互命令前缀。当当前输入以 `/` 开头且不含空格时，底部提示区显示匹配的斜杠命令。

示例：

- `/`：列出所有可用命令。
- `/re`：将建议缩小到 `/resume`、`/refresh` 等命令。
- `/app`：将建议缩小到审批相关命令。

建议列表使用与文件引用相同的键盘模型：

- `Up` 和 `Down` 移动选中的建议。
- 当输入文本不是精确命令时，`Enter` 插入选中的命令并在末尾加空格。
- 当输入已等于支持的命令时，`Enter` 执行该命令。

斜杠建议是 CLI 本地的，执行仍通过现有的 daemon 后端命令处理器完成。
