# 混合输入编辑

混合 CLI 维护一个本地输入缓冲区，而非将提示视为只追加的文本。缓冲区同时跟踪文本和光标位置。

支持的编辑按键：

- `Left` / `Right`：按字符移动光标。
- `Home` / `Ctrl+A`：移动到提示行首。
- `End` / `Ctrl+E`：移动到提示行尾。
- `Backspace`：删除光标前的字符。
- `Delete`：删除光标下的字符。
- `Ctrl+W`：删除上一个单词。
- `Ctrl+U`：清除整个提示。
- `Shift+Enter` 或 `Ctrl+J`：插入换行。
- `Enter`：提交提示（除非选中了补全候选项）。

补全行为使用光标前的文本。在提示中间编辑时，`@` 文件引用和 `/` 斜杠命令均可补全。补全会替换光标前的活动词元，并保留提示的其余部分。

渲染的输入行将光标显示为带样式的 `|` 标记。该标记仅是 UI 显示，不会发送给 daemon。
